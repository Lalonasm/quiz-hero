Q1 : What is JavaScript, and how is it used?

Answer : JavaScript is a high-level, interpreted programming language used primarily to make web pages interactive. It runs on the client side within the browser, enabling dynamic changes to HTML, CSS, and other elements of a web page.

Q2 : What are JavaScript variables, and how do you declare them?

Answer : JavaScript variables are containers for storing data values. You can declare them using var, let, or const, with let and const being block-scoped and preferred in modern JavaScript for safety and consistency.

Q3 : What is an arrow function in JavaScript?

Answer : An arrow function is a shorter syntax for writing functions in JavaScript. It is represented by the => symbol and allows for more concise function expressions, especially useful for callbacks. Arrow functions also have lexical this binding.

Q4 : What is the difference between == and === in JavaScript?

Answer : The == operator checks for value equality after type conversion (loose equality), while === checks for both value and type equality (strict equality). Using === is generally safer as it avoids unexpected type coercion.

Q5 : How does JavaScript handle asynchronous operations?

Answer : JavaScript handles asynchronous operations using callbacks, Promises, and async/await. These methods allow JavaScript to execute code without blocking other operations, crucial for tasks like data fetching or file handling.
